## 学完后能做什么

- 能讲清「模型 × Agent × 指令」三者关系([01-overview.md](01-overview.md))
- 能装好 Claude Code + Codex + 4 件套 skill 工具([02-install.md](02-install.md) + [03-skills.md](03-skills.md))
- 能用 9 步闭环做一个完整的小项目([04-workflow.md](04-workflow.md))
- 能看懂其他同学的 PR
- 有进阶兴趣的能 fork 一个 agent 框架加新 tool([07-agent-dev.md](07-agent-dev.md))

## 验收标准

培训结束后,每个学员必须提交:

1. `claude --version` 和 `codex --version` 输出截图
2. 一个跑通 9 步闭环的 demo PR
3. 一页 A4 的「我的工具选型表」(选了什么 + 为什么)

速查表见 [appendix-cheatsheet.md](appendix-cheatsheet.md)。
